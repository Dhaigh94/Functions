Functions Problem Set

Open the JavaScript console and write the function to see the answer return

  1. Write the function isEven() with a number in the curly brackets and the console will return true if the number is even or false if the number is otherwise. For example isEven(4) will return true because the number is even.
  2. Write a function factorial() which takes a single numeric argument and returns the factorial of that number. For example factorial(10) will return 3628800.
  3. Write a function kebabToSnake() which takes a single kebab-cased string argument and returns the snake_cased version. Basically it will replace a - with _ in a string. For example if you write kebabToSnake("Hello-World") it will return Hello_World.
